It's my final ML project that combines image recognition with a full-stack web application.

## Key Features

- **Image classification** - Upload a photo and get instant waste category identification
- **AI recycling recommendations** - After scanning, get personalized recycling suggestions based on the material type
- **AI chat assistant** - Ask questions about waste disposal and recycling
- **User dashboard** - Track your classification history and see your impact
- **Educational content** - Learn about proper waste disposal methods
- **Blog system** - ContentLayer-based blog with articles about waste management

## Tech Stack

**Frontend**: Next.js 15 with React 19, TypeScript, and Tailwind CSS. **Backend**: Neon PostgreSQL with Drizzle ORM.

**ML & AI Integration**:

- **Garbage Classification Model**: Custom-trained waste classification model deployed on Hugging Face for image recognition
- **Recycling Recommendations**: Powered by GPT-5 nano from Azure OpenAI, providing personalized recycling suggestions based on classified waste types
- **AI Chat Assistant**: Integrated multiple AI SDKs (Azure, Google, OpenAI) for the chat feature

Used React Query for data fetching and NextAuth for authentication. The system combines computer vision (Hugging Face) with generative AI (Azure OpenAI) to create a complete waste management solution.
