Meraih **juara 1** di kompetisi Bootcamp IoT dengan sistem monitoring suhu dan kelembaban real-time ini. Aplikasi ini terhubung ke sensor IoT via protokol MQTT dan menampilkan data live dengan chart. Dilengkapi asisten AI (seperti Alexa atau Home Assistant) yang memungkinkan kamu mengontrol perangkat IoT melalui perintah suara dan pesan MQTT.

## Fitur Utama

- **Monitoring real-time** - Pembacaan suhu dan kelembaban langsung dari sensor
- **Integrasi MQTT** - Koneksi seamless ke data stream sensor
- **Chart interaktif** - Visualisasi tren historis dan real-time
- **Asisten AI** - Asisten yang dikontrol suara untuk mengontrol perangkat IoT melalui perintah MQTT
- **Penyimpanan data** - Simpan dan ambil data sensor historis

## Tech Stack

Dibangun dengan **Next.js 15**, **TypeScript**, dan **MQTT** untuk komunikasi real-time. Menggunakan **Zustand** untuk state management dan **Drizzle ORM** dengan Turso SQLite untuk menyimpan data historis. Asisten AI terintegrasi dengan Azure OpenAI untuk memproses perintah suara dan mengirim pesan kontrol MQTT ke perangkat IoT.
