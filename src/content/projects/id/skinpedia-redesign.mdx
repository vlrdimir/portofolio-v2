Redesign website lama yang sudah ketinggalan zaman menjadi platform modern dan responsif untuk tugas akhir kuliah. Tujuannya sederhana: buat jadi bagus, cepat, dan mudah digunakan di perangkat apapun.

## Perbaikan Utama

- **Desain modern** - Interface yang bersih dan intuitif mengikuti standar web saat ini
- **Mobile-first** - Desain yang sepenuhnya responsif dan bekerja sempurna di ponsel dan tablet
- **Performance lebih baik** - Waktu loading lebih cepat dan interaksi lebih halus
- **Navigasi lebih baik** - Struktur yang disederhanakan dan masuk akal untuk pengguna
- **Accessibility** - Dukungan lebih baik untuk semua pengguna, termasuk yang memiliki disabilitas

## Tech Stack

Dibangun dengan **Next.js** dan framework CSS modern. Fokus pada pembuatan komponen yang dapat digunakan kembali dan codebase yang mudah dirawat. Menggunakan praktik terbaik untuk SEO dan optimasi performance.

> **Catatan:** Repository GitHub untuk proyek ini bersifat private karena diselesaikan sebagai tugas kuliah. Source code tidak tersedia untuk publik.
